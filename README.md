# OSCO CRM

MVP Telegram-based ordering and CRM system for construction store.

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Telegram Bot API
- Render
- Git/GitHub
- REST API

## Features
- Order management
- Payment confirmation workflow
- Admin panel
- Order status tracking
- Backend API integration
- MongoDB database connection
- Real-time order updates

## What I Worked On
- Backend development using Node.js and Express
- Database integration with MongoDB Atlas
- Deployment and service management through Render
- API and Telegram Bot integration
- Troubleshooting backend and deployment issues
- Working with environment variables and server configuration
